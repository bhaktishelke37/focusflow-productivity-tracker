

# FocusFlow — Student Productivity Tracker

A minimal productivity web application designed for students to track **real study time**, manage focused work sessions, and maintain a history of completed tasks.

Unlike a traditional Pomodoro timer, FocusFlow records the **actual time spent on each task**, allowing users to work flexibly while still building productive study habits.

---

## Overview

FocusFlow helps students stay accountable by combining focused work sessions with task-based productivity tracking.

Instead of only counting fixed Pomodoro cycles, users can:

* Start a study session
* Complete a task whenever they finish
* Record the exact time spent on that task
* Take short breaks between work sessions
* Keep track of completed work history

---

## Features

### Task-Based Study Tracking

Enter the task you are currently working on and start a focus session.

### Real-Time Timer

Built-in **25 minute focus timer** following the Pomodoro technique.

### Flexible Task Completion

Finish a task early and save the **actual time spent** instead of waiting for the timer to end.

Example:

* SQL Practice — 18m 42s
* JavaScript Revision — 32m 11s
* DSA Questions — 14m 09s

### Break Mode

Take a **5 minute break** whenever needed without losing workflow.

### Task History

Automatically stores a list of completed tasks with recorded study time.

### Delete Task Option

Remove completed tasks from history when no longer needed.

### Productivity Statistics

Track:

* Total tasks completed
* Total focus time spent studying

### Offline Support

Runs completely offline in the browser with no external dependencies.

## Why I Built This

Most Pomodoro apps focus only on timers.

I wanted to build something more practical for students — a tool that tracks **actual work completed**, not just time running in the background.

The goal was to create a lightweight productivity tracker that combines **focus management** with **real task tracking**.
